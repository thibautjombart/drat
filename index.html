OutbreakResources Drat Repository

Packages for use in outbreaks. Mostly developed as part of the hackout2 hackathon.

Packages

  • apex (v 1.0.1, ): Phylogenetic Methods for Multiple Gene Data
  • epimap (v 0.1, ): Interactive Maps for Epidemiology
  • incidence (v 0.0.0.9000, ): Computation and Visualisation of Incidence from Line Lists
  • mapData (v 0.0.0.9000, ): Tools to help with mapping outbreak data
  • outbreaker (v 1.1-7, ): Bayesian Reconstruction of Disease Outbreaks by Combining Epidemiologic and Genomic Data
  • OutbreakTools (v 0.1-13.999, ): Basic Tools for the Analysis of Disease Outbreaks
  • repijson (v 0.1.1.0, ): Tools for Handling EpiJSON (Epidemiology Data) Files
  • treescape (v 1.8.15, ): Statistical Exploration of Landscapes of Phylogenetic Trees

If drat is not installed, first install from CRAN with

install.packages("drat")

To install packages, run

drat:::add("OutbreakResources")

after which packages can be installed as usual, for example:

install.packages("repijson")
